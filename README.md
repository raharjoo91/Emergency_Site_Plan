# Emergency Field Hospital Site Planning Exercise

A professional facilitator guide for emergency field hospital site planning training exercises.

## Overview

This project contains a comprehensive HTML-based facilitator guide for conducting a 45-minute field hospital site planning exercise. The guide includes:

- Exercise objectives and requirements
- Functional zone specifications
- Three scenario-based reference layouts (Type 1, 2, and 3 hospitals)
- Six additional reference layout images
- Discussion questions and evaluation criteria

## Features

- **Modern UI/UX Design**: Clean, professional interface with contemporary design principles
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Accessibility**: Semantic HTML, proper alt text, and keyboard navigation support
- **Print-Ready**: Optimized styles for printing
- **High-Resolution Layout Display**: Maximum space utilization for layout images (1 layout per space)

## Files

- `Facilitator Guide.html` - Main facilitator guide document
- `Layout 1.png` through `Layout 6.png` - Reference layout images for Appendix 1

## GitHub Deployment

To publish this to GitHub:

1. **Upload Required Files:**
   - `Facilitator Guide.html` (required)
   - `Layout 1.png` through `Layout 6.png` (required - for layout images to display)
   - `README.md` (optional but recommended)

2. **File Structure:**
   ```
   Emergency_Site_Plan/
   ├── Facilitator Guide.html
   ├── Layout 1.png
   ├── Layout 2.png
   ├── Layout 3.png
   ├── Layout 4.png
   ├── Layout 5.png
   ├── Layout 6.png
   └── README.md
   ```

3. **Important:** All files must be in the **root directory** of the repository (same folder level) for the image paths to work correctly.

4. **View Online:** Once uploaded, you can view the guide by navigating to the file in GitHub and clicking "View raw" or by using GitHub Pages (if enabled).

## Usage

Simply open `Facilitator Guide.html` in any modern web browser. The document is self-contained with all styles embedded. **Note:** The layout images will only display if the PNG files are in the same directory as the HTML file.
